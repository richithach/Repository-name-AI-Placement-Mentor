# AI Placement Mentor

An AI-powered career guidance assistant built using Python and Google Gemini API.

## Features

* General Chat
* Interview Questions Generator
* Resume Review
* Skill Roadmap Generator
* Project Suggestions

## Technologies Used

* Python
* Google Gemini API
* Google Colab

## Purpose

AI Placement Mentor helps students prepare for placements by providing interview questions, resume feedback, personalized learning roadmaps, and project recommendations.

## Future Enhancements

* Streamlit Web Application
* PDF Resume Upload
* ATS Score Analysis
* Job Recommendation System
